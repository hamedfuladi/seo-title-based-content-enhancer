== Description ==

The SEO Title-Based Content Enhancer is a powerful WordPress plugin designed to improve your post content's SEO by dynamically inserting text based on the post title. It's a handy tool that helps you create more relevant and keyword-rich content without having to manually update every post.

= Features =

- **Dynamic Text Insertion:** Automatically adds text to the end of your post content based on the post title.
- **Customizable Template:** You can customize the text template using placeholders, including the post title.
- **Exclude Categories and Tags:** Choose specific categories and tags where the dynamic text should not be displayed.
- **Easy-to-Use Admin Interface:** Access the plugin settings through the WordPress dashboard for seamless configuration.

== Installation ==

1. Upload the `seo-title-based-content-enhancer` folder to the wordpress upload plugin page.
2. Activate the plugin.
3. Navigate to 'Settings' > 'SEO Enhancer' to configure the plugin settings.

== Usage ==

1. After activating the plugin, go to the WordPress dashboard and navigate to 'Settings' > 'SEO Enhancer'.
2. In the plugin settings page:
   - Use the text editor to customize the dynamic text template. Click the 'Dynamic Title' button to add the title placeholder ([title]) to the template.
   - Select categories and tags that you want to exclude from displaying the dynamic text.
3. Save your settings.

The plugin will now automatically append the dynamic text, based on the post title, to the end of your post content. Make sure to create compelling and SEO-friendly templates to enhance your content effectively.

== Frequently Asked Questions ==

**Can I use HTML in the dynamic text template?**
Yes, you can use HTML along with placeholders in the text template.

**How do I exclude specific categories from showing the dynamic text?**
In the plugin settings, choose the categories you want to exclude under the "Exclude Categories" section.

**What if I want to exclude specific tags from displaying the dynamic text?**
In the plugin settings, select the tags you want to exclude under the "Exclude Tags" section.
